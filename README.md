# FABPASS-EGatePass-System
Smart eGatePass Management System using Java, XML and Firebase
## Project Workflow

Student → Warden → GFM → HOD → Security

## Technologies Used

- Java
- XML
- Firebase Realtime Database
- Android Studio
- QR Code Scanner

## Features

- Student Request Submission
- Multi-Level Approval
- QR Code Generation
- QR Verification
- Real-Time Firebase Updates
